## Project setup
```
npm install
```

### 启动项目
```
// 开发
npm run dev:%PLATFORM%
// 发布
npm run build:%PLATFORM%
```
具体参照package.json
```
// 微信小程序
npm run dev:mp-weixin

// 百度小程序
npm run dev:mp-baidu

// 支付宝小程序
npm run dev:mp-alipay

// 字节跳动小程序
npm run dev:mp-toutiao

// QQ 小程序
npm run dev:mp-qq
```

在开发工具导入dist\dev下对应平台的包即可预览
